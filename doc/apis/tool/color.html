色彩

color 

var rgbaArray = $$.color(param);

传递一个合法的CSS颜色字符串，统一转换为rgba(x, x, x, x)格式的数据，返回数组[r, g, b, a]。

getColors 

var colors = $$.getColors(num,range,baseColor);

传递整数,色彩范围和基准颜色，返回一组色彩。

num为最终返回的颜色个数。

range为色彩范围，是一个数组，取值范围0-360。

baseColor可选参数，为基准颜色，是一个数组，包含rgb三个值（0-255），返回的颜色数组都将以该色调为基准。 

var colors = $$.getColors(15, [0, 270], [20, 156, 203]);

另外，如果你只是想要一组颜色，可以直接指定颜色数目，返回随机色彩： 

var colors = $$.getColors(15);
