XHR 请求

get 

var get=$$.get(header,timeout);

获取get对象，需要传递二个参数，当然，都是可选的：

  • 1.header：是一个json类型的键值对，用于设置请求头；
  • 2.timeout：超时时间，请求超时会触发错误回调。
get(url[, callback[, errorback]]);

传递请求地址，回调和错误回调都可选，回调参数是一个json，分别表示：

  • 1.response:响应数据；
  • 2.status：状态码；
  • 3.header：响应头信息。

错误回调参数也是一个json，分别表示：

  • 1.type：表示错误类型，"error"或"timeout"。

返回请求对象(你可以继续发送请求)。

post 

var post=$$.post(header,timeout);

参数说明和上面一样（包括返回数据格式），接着发送请求： 

post(url, param[, callback[, errorback]]);

post请求可以带一个参数，数据类型是json。
