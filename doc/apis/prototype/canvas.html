Canvas 图形接口
  • 1.painter.beginPath()：起始一条路径，或重置当前路径；
  • 2.painter.moveTo(x, y)：把路径移动到画布中的指定点(x, y)，不创建线条；
  • 3.painter.closePath()：创建从当前点回到起始点的路径；
  • 4.painter.lineTo(x, y)：添加一个新点(x, y)，然后在画布中创建从该点到最后指定点的线条；
  • 5.painter.clip()：从原始画布剪切任意形状和尺寸的区域；
  • 6.painter.quadraticCurveTo(cpx, cpy, x, y)：创建二次贝塞尔曲线；
  • 7.painter.bezierCurveTo(cp1x, cp1y, cp2x, cp2y, x, y)：创建三次贝塞尔曲线；
  • 8.painter.arc(x, y, r, sAngle, eAngle, counterclockwise)：创建弧/曲线；
  • 9.painter.arcTo(x1, y1, x2, y2, r)：创建两切线之间的弧/曲线；
  • 10.painter.isPointInPath(x, y)：如果指定的点(x, y)位于当前路径中，则返回 true，否则返回 false。

clip

该方法表示从原始的画布中裁剪一个区域，一旦裁剪完成了，后续绘制的图形都会被限制在裁剪的区域中。

解除clip限制

如果你想取消前一步执行clip()产生的绘制区域的限制，可以借助save和restore来实现： 

painter.save();
...
painter.clip();
// 存在限制
...
painter.restore();
// 已经解除了限制
...

裁剪区域的确定

默认的区域是整个canvas（或者说没有裁剪区域），你可以通过创建路径后调用clip方法，裁剪区域就是前面路径表示的区域。
