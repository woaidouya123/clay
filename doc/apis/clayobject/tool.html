有益的方法

主要是一些非基础但是在绘图中会经常使用到的高效方法。

size 

var size=clayObject.size(type);

返回clay对象维护的第一个结点的宽和高，格式如下： 

{
    width:number,
    height:number
}

其中，可以传递一个参数type，也可以不传使用默认值，具体可以传参数如下：

  • 'content'：计算的是内容的宽高；
  • 'padding'：计算的是内容+内边距的宽高；
  • 'border'：计算的是内容+内边距+边框的宽高（缺省值）；
  • 'scroll'：计算的是滚动的宽（不包括边框）的宽高。
