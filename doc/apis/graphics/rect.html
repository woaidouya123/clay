矩形 rect

绘制矩形的时候，考虑到各种情况，我们把矩形看成一个有宽度的线条，通过设置线条的宽，和长度等来绘制不同的矩形，这是在参数设置上的想法，具体绘制的依旧是矩形，不是线条。

Canvas2D 

var rect = $$.canvas.rect('canvas')
    // 设置矩形木棒的粗细
    .setSize(size)
    // 设置矩形方向类型
    .setType(type[, cx, cy]);

上面的方法setType表示设置矩形方向，可以设置参数：

  • 1.垂直或水平 "LR"、"RL"、"TB"、"BT"
  • 2.任意角度 (deg,cx,cy)，deg表示初始角度，(cx,cy)表示旋转圆心

如果选择第一种，只需要一个参数，比如： 

rect.setType('LR');

这表示绘制的木棒是水平从左到右的。

如果选择第二种，就需要三个参数，这是为了方便绘制旋转的矩形，旋转指的是矩形整体旋转。 

rect(x, y, length[, deg]).fill();

其中（x,y）表示木棒起点坐标，length表示木棒长度，最后一个deg表示旋转角度，只有上面setType设置第二种的时候才需要传递。

木棒实际旋转角度是前面setType设置的deg加上这里的deg之和。

SVG 

var rect = $$.svg.rect()
    // 设置矩形木棒的粗细
    .setSize(size)
    // 设置矩形方向类型
    .setType(type[, cx, cy]);

参数说明和上面的一样。 

$$('path').attr('d', rect(x, y, length[, deg]))
    .css({
        'key':'value',
        ...
});
