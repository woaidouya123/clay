文字 text

Canvas2D 

var text = $$.canvas.text('canvas')
    // 设置对齐方式（可选）
    .setAlign(horizontal, vertical)
    // 设置字体大小，传递一个数字（可选）
    .setSize(size)
    // 设置字颜色（可选）
    .setColor(color);

对齐方式表示字的位置（设置错误或缺省值都是居中对齐）：

  • 1.水平对齐方式horizontal："left","right","center"；
  • 2.垂直对齐方式vertical："top","bottom","middle"。
text(x, y, textstring[, deg]).fill();

在（x,y）处绘制文字textstring,deg可以不传递，除非你希望绘制的文字旋转一定角度。

SVG 

var text = $$.svg.text()
    // 设置对齐方式（可选）
    .setAlign(horizontal, vertical)
    // 设置字体大小，传递一个数字（可选）
    .setSize(size)
    // 设置字颜色（可选）
    .setColor(color);

参数说明和上面一样。 

var clayObject=text(x, y, textstring[, deg]);

这里返回的是一个clay对象，维护着一个文字标签text，你需要主动把该标签追加到页面中去。
